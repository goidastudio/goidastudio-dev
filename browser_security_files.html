Важное предупреждение о безопасности

Приложение Ghost Browser позволяет загружать файлы из интернета, но мы должны предупредить вас о потенциальных рисках.

Потенциальные опасности:

  • Файлы могут содержать вирусы, трояны или другое вредоносное ПО
  • Некоторые файлы могут повредить ваше устройство или украсть ваши данные
  • Исполняемые файлы особенно опасны
  • Файлы могут содержать неожиданное или нежелательное содержимое

Наша позиция:

Разработчики Ghost Browser не несут ответственности за:

  • Любые повреждения устройства, вызванные скачанными файлами
  • Потерю данных в результате использования скачанных файлов
  • Любые другие последствия использования загруженных файлов

Загружая файлы через наше приложение с вашего явного согласия ("я даю согласие"), вы полностью принимаете на себя все риски, связанные с использованием этих файлов.

Рекомендуем:

  • Загружать файлы только из проверенных источников
  • Проверять все файлы антивирусом перед открытием
  • Быть осторожными с файлами, требующими особых разрешений
